# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML5** and **CSS3**. This portfolio showcases my education, technical skills, projects, certifications, and contact information in a clean and professional design.

## 🚀 Features

* Responsive design for desktop and mobile devices
* Modern dark theme with glassmorphism effects
* Smooth scrolling navigation
* Professional "About Me" section
* Skills showcase
* Project showcase
* Certificates section
* Contact information
* Resume download button
* Animated typing effect

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts (Poppins)

## 📂 Sections

* Home
* About Me
* Career Objective
* Education
* Skills
* Projects
* Certificates
* Contact
